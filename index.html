Project description.

Vaaclipse is the way to use the power of the Eclipse e4 platform in web applications development. Vaaclipse uses the vaadin to move e4 to Web and provides eclipse based platform for building server side web applications.

About e4

E4 is the future of the Eclipse Platform. E4 provides the great concepts thats helps you build modern modular and modeled applications. The platform design allows use different UI toolkits for GUI presentation. The appropriate GUI presentation layer is called renderer. Eclipse RCP applications (and Eclipse itselft) use the SWT renderer. Except SWT there are, forexample, JavaFX renderer by Thomas Schindl called efxclipse allowing you build eclipse applications using JavaFX (the modern java UI toolkit). The vaaclipse project provides you the vaadin renderer for e4 platform.

The e4 project description on eclipse.org: http://www.eclipse.org/e4/

A lot of usefull publications and tutorials about e4 can be founded on www.vogella.com

About vaadin

Vaadin is the Java framework for building modern web applications. It does web development very easy. Together with e4 platform vaadin helps you to build next gen web applications in modular and modeled way.
